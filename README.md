# 🏏 IPL Analysis Dashboard (2008–2025) | Power BI

An interactive Power BI dashboard that provides comprehensive insights into the Indian Premier League (IPL) from 2008 to 2025.
The dashboard enables users to analyze team performance, player statistics, season-wise results, and tournament records through dynamic visualizations.

---

## 🚀 Features

- 📅 Season-wise analysis using interactive slicers
- 🏆 Champion & Runner-Up details for each IPL season
- 📈 Overall tournament statistics
  - Total Matches
  - Total Fours
  - Total Sixes
  - Total Venues
  - Centuries
  - Half Centuries
- 🟠 Orange Cap analysis
  - Highest Run Scorer
  - Total Runs
  - Team
- 🟣 Purple Cap analysis
  - Highest Wicket Taker
  - Total Wickets
  - Team
- 💥 Player Statistics
  - Most Fours
  - Most Sixes
- 📋 Season Points Table
- 🎨 Interactive and responsive dashboard design

## 🎯 Business Objective

The objective of this dashboard is to transform raw IPL data into meaningful insights that help users:

- Compare team performances across seasons
- Analyze player achievements
- Track tournament records
- Explore batting and bowling statistics
- Make data-driven sports analysis
